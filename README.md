# Random Quote Generator

Раду Екатерина

## Описание
Это графическое приложение, которое позволяет генерировать случайные цитаты из предопределенного списка, сохранять историю, фильтровать цитаты по автору и теме, а также добавлять новые цитаты.

## Использование
- Нажмите «Сгенерировать цитату» для получения случайной цитаты.
- Нажмите «Показать историю» для просмотра всех сгенерированных цитат.
- Используйте «Фильтр» для поиска цитат по автору и теме.
- Нажмите «Добавить цитату» для добавления новой цитаты.

## Тесты
- Проверка генерации случайных цитат.
- Проверка сохранения и загрузки истории.
- Проверка фильтрации по автору и теме.
- Проверка добавления новых цитат.
